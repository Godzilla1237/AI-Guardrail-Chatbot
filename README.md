# AI Guardrail Chatbot

This project is a Streamlit-based AI chatbot safety system developed for the CARA Summer Internship.

## Features

- Toxicity detection
- PII detection
- Prompt injection detection
- Unsafe message blocking
- CSV-based logging
- Admin dashboard with flagged message statistics

## Tech Stack

- Python
- Streamlit
- Pandas
- Regex
- CSV Logging

## How to Run

```bash
pip install streamlit pandas matplotlib
streamlit run app.py
